# Third-Party License Pointers

本目录用于存放第三方许可证副本（如你后续希望将许可证文本一并纳入仓库）。

当前项目第三方组件许可证来源如下：

- FFmpeg（BtbN GPL 构建）：见 `deps/ffmpeg-master-latest-win64-gpl/LICENSE.txt`
- yt-dlp（Unlicense）：见上游仓库 `https://github.com/yt-dlp/yt-dlp/blob/master/LICENSE`
- whisper.cpp（MIT）：见上游仓库 `https://github.com/ggml-org/whisper.cpp/blob/master/LICENSE`

如果你计划发布带二进制的发行包，建议将相关许可证文本复制到本目录并随发行包附带。
